Challenge Data Science – Alura Store

Este proyecto analiza los datos de cuatro tiendas de Alura Store para ayudar al Sr. Juan a decidir cuál tienda debería vender, basándose en ingresos, ventas, valoraciones y costos de envío.

Datos utilizados

Se cargaron cuatro archivos CSV desde GitHub, cada uno representando una tienda distinta. Se trabajó con:

Precio

Categoría del producto

Valoración (Review)

Costo de envío

Productos vendidos
Análisis realizado

- Ingresos por tienda

Se sumaron los valores de la columna Precio.
Resultado: La Tienda 1 generó los mayores ingresos.

- Ventas por categoría

Se contó cuántos productos se vendieron en cada categoría por tienda.

- Valoración promedio

Todas las tiendas tienen una valoración cercana a 4, mostrando buena satisfacción general.

- Productos más y menos vendidos

Se identificaron mediante value_counts().

- Envío promedio

Se calculó el costo de envío promedio por tienda.
Resultado: La Tienda 1 tiene el envío más costoso; la 4, el más económico.
